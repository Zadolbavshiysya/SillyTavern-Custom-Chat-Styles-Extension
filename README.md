# SillyTavern-Custom-Chat-Styles-Extension
Стили для офтрмления текста с помощью тегов. Включает меню настроек цвета и инжект системного промпта. С помощю регулярных выражений работают как с []...[/], так и с <>...</> 

На данный момент существуют теги:

* - [chat]Имя: Сообщение[/chat] - Для чата стримов и подобного.*
<img width="800" alt="изображение" src="https://github.com/user-attachments/assets/62bd0996-fc8a-436b-9519-085250e0943a" />

* - [phone]Имя: Сообщение[/phone] - Для сообщений в месенджерах.*
<img width="800" alt="изображение" src="https://github.com/user-attachments/assets/5a59507c-3c1e-407c-b7f2-2b61701e138f" />


Причём, если имя вашей персоны совпвдает с именем в шаблоне "Имя: Сообщение" то сообщения отобразятся справа. (В примере выше я как раз переключилась на другую персону, а ниже снова на Сета)
